# Python менен программалоого киришүү

Бул эксперименталдык репозиторий башталгыч балдарга (5–6-класстар) Python аркылуу программалоонун негиздерин кыргыз тилинде үйрөтүү үчүн түзүлгөн.
Эгер сиз бул долбоорго салым кошсоңуз, мен ыраазы болот элем.

## Кантип колдонуу керек
- **boluktor/** папкасында ар бир бөлүм өзүнчө Markdown файл болуп жайгашкан.
- **surottor/** — сүрөттөр үчүн (азырынча бош).
- Кандай болбосун Python 3.10+ версиясы жарайт. Онлайн үчүн: Google Colab же сайттарты колдоно аласыз.
