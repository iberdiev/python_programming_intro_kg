# Бөлүк 5: Маалымат түрлөрү

Негизгиси:
- **int** — бүтүн сан: `5`
- **float** — ондук сан: `3.14`
- **str** — сап: `"салам"`
- **bool** — логикалык: `True` же `False`

```python
a = 5
b = 3.5
c = "python"
d = True
print(type(a), type(b), type(c), type(d))
```

**Тапшырма:** Ар бир түргө мисал ойлоп табыңыз.
