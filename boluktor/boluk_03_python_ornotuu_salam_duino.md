# Бөлүк 3: Python орнотуу жана «Салам, дүйнө!»

Python'ду орнотуу:
- [python.org] сайтынан жүктөп орнотуңуз.

Алгачкы программа:
```python
# Экранга текст чыгаруу
print("Салам, дүйнө!")
```

**Эмне болду?** `print(...)` — экранга маалымат чыгарат.

**Тапшырма:** Өз атыңыз менен саламдашуучу текст чыгарыңыз.
