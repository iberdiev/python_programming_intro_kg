# Бөлүк 20: Ката (error) деген эмне?

Түрлөрү:
- **Синтаксистик ката** — туура жазылбаган код
- **Аткаруудагы ката** — программа иштеп жатып токтойт
- **Логикалык ката** — программа иштейт, бирок натыйжа туура эмес

```python
# Мисал: ValueError
try:
    jash = int(input("Жаш: "))
    print("Кийинки жыл:", jash + 1)
except ValueError:
    print("Сан жазыңыз.")
```

**Тапшырма:** Нөлгө бөлүү катасын кармап көрүңүз.
