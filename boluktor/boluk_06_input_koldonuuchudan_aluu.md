# Бөлүк 6: Колдонуучудан маалымат алуу

`input()` — колдонуучудан текст алат.
```python
aty = input("Атыңыз ким? ")
print("Салам,", aty)
```

Санды алуу үчүн айландырабыз:
```python
jash = int(input("Жашыңыз: "))
print("Бир жылдан кийин:", jash + 1)
```

**Тапшырма:** Колдонуучудан 2 сан алып, суммасын чыгарыңыз.
